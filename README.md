# Single Page App – Notes

## Run locally (no build step)

    npx serve .
    python3 -m http.server 8000

Open http://localhost:8000 in your browser.

## Add Vite (bundler)

Vite works as a "compiler": you write code in `src/`, it produces an
optimised `dist/` folder ready to upload to a server.

**1. Initialize the project:**

    npm init -y

**2. Install Vite:**

    npm install --save-dev vite

**3. Add scripts to `package.json`:**

![](package.json)

**4. Start the dev server:**

    npm run dev

**5. Build for deployment:**

    npm run build

The `dist/` folder is what you upload to the server — not your `src/`.

## Project structure

```
single_page_app_example_1/
├── index.html          ← app shell (stays at root for Vite)
├── src/
│   ├── app.js          ← entry point: registers routes
│   ├── router.js       ← reusable hash-based Router class
│   ├── css/
│   │   └── style.css
│   └── pages/
│       ├── home.js
│       ├── about.js
│       ├── contact.js
│       └── notfound.js ← catch-all 404
└── dist/               ← generated by `npm run build` (do not edit)
```


## Adding a new page

1. Create `src/pages/mypage.js` with a `render()` method.
2. Import it in `src/app.js` and call `router.addRoute('/mypage', myPage)`.
3. Add `<a href="#/mypage">My Page</a>` to the nav in `index.html`.
